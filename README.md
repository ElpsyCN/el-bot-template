# el-bot-template

本仓库为包含 [mirai](https://github.com/mamoe/mirai) 与 [el-bot](https://github.com/ElpsyCN/el-bot) 的基础可运行模版。

你需要确保你已拥有 [Java](https://www.java.com/zh_CN/) 与 [Node.js](https://nodejs.org/zh-cn/download/) 环境。

## 安装 mirai

```sh
npm install:mirai
```

```sh
npm install

# 运行 mirai 和 bot
npm run start

# 运行 mirai
npm run start:mirai

# 运行 bot
npm run start:bot
# yarn bot
```
